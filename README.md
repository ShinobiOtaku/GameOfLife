# Conway's Game of Life

Basic implementation of Conway's game of life, in F#.

Inputs from `'sample_input.txt'` in `src/`, outputs to console.

https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life

## Running

```bash
cd src
dotnet run
```

## Running tests

```bash
cd test/badlife.tests
dotnet run
```
